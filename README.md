### 📋 Tabla de Funciones
| Función | Responsabilidad |
| :--- | :--- |
| `LeerOpcion` / `LeerNoches` | **UI:** Captura y valida datos del usuario. |
| `CalcularSubtotal` / `CalcularImpuesto` | **Lógica:** Realiza cálculos matemáticos puros. |
| `ProcesarReserva` | **Orquestación:** Coordina el flujo entre captura, cálculo y salida. |
| `MostrarResumen` | **UI:** Presenta los resultados finales formateados. |
